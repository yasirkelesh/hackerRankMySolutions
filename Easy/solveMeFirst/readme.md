This function computes the sum of two integers.

## Example

```plaintext
a = 7
b = 3
Return: 10
Function Description
Complete the solveMeFirst function in the editor below.

Parameters:
int a: The first value.
int b: The second value.
Returns:
int: The sum of a and b.
Constraints
1 ≤ a, b ≤ 1000
Sample Input
a = 2
b = 3
Sample Output
5
Explanation
